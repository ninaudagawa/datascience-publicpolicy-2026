# Figure & Visualization Style Guide

This guide defines styling, formatting, and export standards for visualizations generated in this project.

---

## 1. Visual Design & Theme Guidelines

All generated plots and figures must adhere to a clean, readable design:
- **Typography**: Use standard clean fonts (e.g., Inter, Outfit, or sans-serif).
- **Color Palette**: Avoid default primary colors. Use curated HSL tailored or muted, cohesive palettes matching the document style.
- **Labels**: Every figure must have a clear title, labeled axes with units, and a legend if multiple series are plotted.

---

## 2. Export & Storage Conventions

- **Storage Path**: Save generated visualizations in `reports/figures/` or `reports/`.
- **Formats**: Export plots in PNG format for presentations or SVG format for documents and white papers.
- **File Naming**: Use descriptive, lowercase names with underscores (e.g., `gdp_growth_trend.png`).
